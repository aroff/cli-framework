# `basic_cli`

Minimal **cli-framework** usage: two commands (`hello`, `increment`) and a small read-eval loop (not full argv parsing).

## Run

From the repository root:

```bash
cargo run --example basic_cli
```

Commands are entered at the `>` prompt (e.g. `hello Alice`, `increment`, `quit`).
